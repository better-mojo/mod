# Mod(Mojo Dep)

- `Mod` = `Mojo Dep`.
- `Mod` is `Mojo Package Manager Tool`.
- like:
    - Rust: `Cargo`
    - Python: `Pip/Poetry/PDM`
    - Go: `Dep + Vendor`

## Quick Start

- install:

```ruby

# all Platform
pip install mojo-dep

# MacOS
brew install mojo-dep

# Linux
apt install mojo-dep

# Windows
pip install mojo-dep

```

### Manual

- `mojo mod -h`

```ruby

mod -v

# 帮助:
mod --help


```
